# Kinofan

Front-end додаток для пошуку фильмів та серіалів. Демо: https://kinofan.netlify.com/

# Стек технологій:

-   React + React Hooks
-   Firebase (авторизація)
-   Redux Toolkit, RTK Query
-   Scss, CSS modules
-   Formik

UI:
-   Lightgallery 
-   Swiper
-   react-loading-skeleton
-   react-range
-   react-select
